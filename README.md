# Exercícios Docker - Orion Bootcamp

Primeiro exercício do Orion Bootcamp - New Rizon.

Exercícios práticos de Docker organizados por branches.

## Estrutura

```
exercicios-orion/
├── README.md
├── .gitignore
└── docker/
    ├── README.md (instruções detalhadas)
    ├── exercicio-1/ (Node.js)
    ├── exercicio-2/ (Python Flask)
    ├── exercicio-3/ (Docker Compose)
    └── exercicio-4/ (Volumes)
```

## Pré-requisitos

- Docker Desktop instalado
- Git instalado
- Editor de código

## Como Usar

```bash
# Clone o repositório
git clone https://github.com/GaabrielCH/exercicios-orion.git
cd exercicios-orion

# Acesse a pasta docker para ver as instruções
cd docker
cat README.md
```

## Branches

- **main**: Código completo
- **develop**: Desenvolvimento
- **exercicio-1 a 4**: Branches dos exercícios
